# Trading Bot - Binance Futures Testnet

A simplified Python CLI application that places Market and limit orders on Binance Futures Testnet 
(USDT-M), with input validation, structured logging, and clean console output.


## Setup 

1. Register for a Binance Futures Testnet account at https://testnet.binancefuture.com
2. Generate an API Key and Secret from the testnet dashboard.
3. Clone/download this project and create a virtual environment: 
4. Install dependencies:
